2) 영어 original 결과물 생성 후, 각 단계마다 한국어로 번역하는 서브에이전트가 작동.

   현재 워크플로우에 한국어 번역 담당(Task Management / agent-teams / Agent Swarm /
   orchestrator / sub agents / skills / hooks / commands / task verification)이
   없으면 즉시 신설 제안.
