1. 먼저 /prompt/prd-research/ 의 현재 구조와 확장 규칙 문서를 확인한다.

   - 차수 태깅: 모든 파일 메타데이터에 "조사 차수 = 2차" 명시.

   - 메타데이터 필수: 조사 차수 / 조사 축 / teammate 식별 / 생성 시점 /
                      원본 질문 / 근거 출처 / 로컬 실행 태그(LOCAL-OK/PARTIAL/BLOCKED).
